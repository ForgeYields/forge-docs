# 🖖 Introduction

ForgeYields is a non-custodial, automated allocator that hunts the best incentivized yields across chains. It deploys and rotates liquidity dynamically, optimizing for real risk-adjusted returns with zero user effort.

<figure><img src=".gitbook/assets/header.png" alt=""><figcaption></figcaption></figure>



**What you get**

* **Higher net yields** via automated, cross-chain incentive rotation.
* **True non-custodial control:** you hold fyTokens, never IOUs.
* **Full transparency:** every allocation and AUM update logged in the Atomic Transparency Ledger.
* **No lockups:** mint once, auto-compound, redeem whenever you want.
* **Clear performance:** live Projected vs Realized yield tracking.
